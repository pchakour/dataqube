# Transformers

Transform your data using one of the following plugins:

| Plugin | Description |
|---|---|
| [add_field](./add_field.md) | 
  This plugin allow you to add a new field in your record


  ```
  - tag: ADD_FOO_FIELD
    transform:
      - type: add_field
        name: 'foo'
        value: 'fee'
  ```
